# About

What is my IP?

CLI utility for the [`public-ip`] crate.

[`public-ip`]: https://crates.io/crates/public-ip

# Usage

```text
$ whatismyip -h
What is my IP?

Usage: whatismyip

Options:
  -h, --help     Print help
  -V, --version  Print version
```

```text
$ whatismyip -V
whatismyip 0.1.0
```
